The lever rule in thermodynamics

Ke Gao
2023-04-16
mineral, thermodynamics

Derivation of the lever rule in thermodynamics in mineralogy.

Question

Suppose a mixture consisting of the components A and B has the composition \(x^b_B\). Suppose further that at some P and T, A and B react with each other forming the phases \(\alpha\) and \(\beta\), with compositions \(x^\alpha_B\) and \(x^\beta_B\), respectively. The question to be answered is: what are the molar proportions of the two phases.

Answer

We know

\[\begin{equation}\label{xbB} x^b_B = \frac{n^b_B}{n^b_A + n^b_B} \end{equation}\]

and

\[\begin{equation}\label{nbB} n^b_B = n^\alpha_B + n^\beta_B \end{equation}\]

and

\[\begin{equation}\label{nbA} n^b_A = n^\alpha_A + n^\beta_A \end{equation}\]

, if we give eq \(\eqref{nbB}\) and \(\eqref{nbA}\)to eq\(\eqref{xbB}\) we will have

\[\begin{equation}\label{xbB2} x^b_B = \frac{n^\alpha_B + n^\beta_B}{n^\alpha_B + n^\beta_B + n^\alpha_A + n^\beta_A} \end{equation}\]

For the same reasonï¼Œ we have

\[\begin{equation} x^\alpha_B =\frac{n^\alpha_B}{n^\alpha_A + n^\alpha_B} \end{equation}\]

and

\[\begin{equation} x^\beta_B =\frac{n^\beta_B}{n^\beta_A + n^\beta_B} \end{equation}\]

, so \(n^\alpha_B = x^\alpha_A(n^\alpha_B + n^\alpha_B)\) and \(n^\beta_B = x^\beta_B(n^\beta_A + n^\beta_B)\) , and if we give them to eq\(\eqref{xbB2}\), we have

\[\begin{equation}\label{eq} x^b_B(n^\alpha_A + n^\alpha_B) + x^b_B(n^\beta_A + n^\beta_B) = x^\alpha_B(n^\alpha_A + n^\alpha_B) + x^\beta_B(n^\beta_A + n^\beta_B) \end{equation}\]

. If we divide both sides of the eq\(\eqref{eq}\) by \((n^\beta_A + n^\beta_B)\) at the same time, we can say

\[\begin{equation}\label{eq8} x^b_B\frac{(n^\alpha_A + n^\alpha_B)}{(n^\beta_A + n^\beta_B)} + x^b_B = x^\alpha_B\frac{(n^\alpha_A + n^\alpha_B)}{(n^\beta_A + n^\beta_B)} + x^\beta_B \end{equation}\]

Rectifying the eq\(\eqref{eq8}\) yields that

\[\begin{equation} \frac{(n^\alpha_A + n^\alpha_B)}{(n^\beta_A + n^\beta_B)} = \frac{(x^\beta_B - x^b_B)}{(x^b_B - x^\alpha_B)} \end{equation}\] .
