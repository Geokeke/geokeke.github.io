给用Chrome或火狐的你，推荐个神器

Posted on 2018-10-29 Edited on 2018-10-31

用了一段时间油猴脚本，说说我留下的几款。 ## 油猴脚本是什么，什么人会需要？

一句话讲，油猴脚本是能解决很多痛点问题的浏览器上的扩展程序。

我觉得用 Chrome 或火狐浏览器的人都会用到油猴脚本，甚至爱不释手（如我）。

如果你不用 Chrome 或火狐，那你该换浏览器了😄。

如何使用？

要使用油猴，需要两个东西，一个是油猴脚本管理器，一个是相应的油猴脚本。

最初的油猴脚本管理器是火狐的扩展程序 Greasemonkey（GM），而 Tampermonkey（TM） 是 Chrome 上的后来居上者，随后 TM 也 Microsoft Edge、Safari、Opera Next、Firefox、Dolphin Browser、UC Browser 等浏览器平台推出，你可以在 软件官网 下载到对应版本，我用的就是这款。此外还有 Violentmonkey，俗称“暴力猴”，听名字大家就可以猜到，这个管理器可以实现更多增强型的功能。

Tampermonkey 可以直接从 Chrome 或火狐的扩展程序商店安装。如果不能，推荐一个叫 Crx4Chrome 的网站，可以直接下载 Chrome Store 插件。在 Crx4Chrome 搜索并下载 Tampermonkey 插件 到本地之后，再打开 Chrome 浏览器「扩展程序」页面，将下载的 crx 文件拖拽到页面即可完成安装。

说完了管理器，而油猴脚本要从哪里安装呢？推荐一个渠道，Greasy Fork，需要什么脚本都可以去这里搜。

油猴脚本可以让视频网站强制使用 html5 播放器、突破百度网盘限速、各大平台在线 VIP 视频免费看、各大平台付费音乐下载、一些网站页面样式优化等，可以说是给浏览器开挂了。

我在使用的脚本

在尝试了许多款油猴脚本后，我留下了以下几款：

  1. 反广告利器 Anti-Adblock Killer 浏览页面，广告越来越多了。这款脚本可以屏蔽页面的广告，包括百度、各种视频网站。

  2. 从豆瓣直达资源的 Douban Download Search（支持 https） 如果你也是豆友，喜欢在豆瓣看书籍、电影等的评分，那这款脚本简直不能太好用。 比如我，听说了一部电影后，我会先在豆瓣看看评分和评论，如果OK，就会找资源来看。 而这个脚本解决的，就是从豆瓣到找到资源这一步。安装了这个脚本后，会在页面上出现豆瓣电影、图书、音乐的下载搜索链接，一点即达，省去了不少复制粘贴和搜索的时间。 这是没安装这个脚本的豆瓣页面： 下图是安装此插件的效果，可以看到，文字部分出现了很多链接： 是不是很方便！

  3. 从豆瓣电影看 IMDb 和烂番茄评分的MoreMovieRatings 我选电影的时候，会综合豆瓣评分、IMDb评分和烂番茄新鲜度。但是同一部电影搜索三次，时间吗长了也是个麻烦的事儿。有了这个插件，在一个豆瓣页面，就可以看到三家评分，很爽有木有！

  4. 解决百度云大文件下载限制 在网页上找到百度云的资源，如果文件比较大，百度云就会要求启动客户端。有了这个插件，就可以直接下载了，如果配合IDM，速度也会相当有保障。

  5. 为什么你们就是不能加个空格呢？ 如果你跟我一样，每次看到网页上的中文字和英文、数字、符号挤在一块，就会坐立难安，忍不住想在它们之间加个空格。这个 脚本正是你所需要的，它会自动替你在网页中所有的中文字和半形的英文、数字、符号之间插入空格。

  6. 一键下载文献的Sci-hub button Sci-hub 是科研利器，这个插件让 Sci-hub 更好用了。安装此插件后，可以直接在谷歌搜索或其他学术搜索页面点击下载文献。

  7. 显示当前网站所有可用脚本的 Userscript+ 当你浏览网页的时候, Userscript+ 会从右下角自动为你推荐适用于当前页面的油猴脚本，并且可以一键安装指定的脚本。

    很多时候，我们并不知道一些网站是否有用户提供用来优化页面的脚本，而 Userscript+ 就能帮你自动寻找适用的 UserJS，并默认按照评分高低排序推荐给你。

  8. 看真正想看的微博 Yet Another Weibo Filter 微博的广告和排序方式有时让人迷茫。这个插件可以还你一个纯净版的微博。 主要功能：

    • 根据关键字、作者、来源等隐藏、折叠或高亮微博；使用拖拽轻松定义过滤规则；
    • 屏蔽推广、粉丝头条、投票、好友赞过、抢红包、爱问医生等各种微博；防刷屏、版聊； -自动加载和过滤新微博，白名单新微博桌面提示；禁用首页下方的延迟加载；
    • 清理版面上的各种模块、图标、小红点，去广告；过滤热门话题；使用方形头像；
    • 合并左右边栏的双栏模式，加宽加大的微博宽度和微博字号，自定义微博字体；
    • 去除微博间的空白，缩小微博下操作按钮距离，重新安排他们的顺序；
    • 自动检查首页上是否遗漏了特定用户的最新微博，不错过他的任何一条消息；
    • 展开左栏消息分组；让各种浮动的模块回到原位；统一各种页面的边栏位置；
    • 查看原图，展开 t.cn 短网址；清除发布框默认话题，禁止自动关注话题主持人；
    • 微博作者和正文不拆行；按字数自动展开或隐藏长篇幅的微博，禁用微博换行；
    • 设置网页模板，自定义半透明背景色，深色导航栏，经典导航栏布局；
    • 方便的新功能更新提醒；设置备份，以及在导入时提示设置最新功能；
    • 使用本机时区显示微博上的各种时间（针对非东八区用户）；
    • 正常大小的微博缩略图尺寸，显示在微博下面的发布时间和来源（针对 2016 新版）

今天就推荐以上 8 个油猴脚本啦，都是我常用的，希望大家喜欢。如果你有喜欢的油猴脚本，也可以介绍给我呀。
