第一篇博客

Ke Gao
2017-05-30

这是我的第一篇博客。

第一个建议我开博客的人是 cycleuser，那时我刚在 Github 上创建一个账号，打算将每日感恩和成功日记记录在上面（虽然由于步骤繁琐并没有坚持很久）。他当时在管理一个叫做 Geopython 的项目，我 star 了。然后他看见了我的每日感恩和成功日记，建议我尝试一下 Pelican 或者 Hexo + Github Pages 制作个人站点。

但是没有刚需，一直没有尝试。最近决定搭一个自己的博客是因为，优达学城的数据分析师优等生导师把我拉进了他们小组，于是之前我一个人单打独斗，变成和十几个小伙伴齐头并进了。这个小组有个额外任务，就是搭建自己的学习博客，把学到的知识分享给别人，也让自己学得更扎实。

导师再次推荐了 Hexo + Github Pages 的方案，本次博客的搭建就是采用此方案。

虽然还没有配置好，但是自己看文档，找教程，配置 Github Pages，配置 Hexo，购买域名 Gaoke.online，最终有了一个独立域名的博客，还是有点成就感的。

以后经常在这里敲敲打打吧，毕竟我是个输出需求比较旺盛的人哈哈。
